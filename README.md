# Trip-Agency – Web Application Full Stack (PHP & MySQL)

Applicazione web full stack sviluppata in PHP e MySQL per la gestione di un’agenzia di viaggi.
L’applicazione consente di gestire clienti, pacchetti viaggio e prenotazioni tramite operazioni CRUD e database relazionale.

---

## Funzionalità

- Gestione dei pacchetti viaggio (creazione, modifica, eliminazione, visualizzazione)
- Gestione dei clienti
- Gestione delle prenotazioni associate ai clienti e ai pacchetti
- Relazioni tra tabelle (clienti, destinazioni, prenotazioni)
- Operazioni CRUD complete
- Validazioni lato server
- Interfaccia web semplice e responsive
